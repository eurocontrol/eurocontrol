# Return the corresponding IATA season for a date

Return the corresponding IATA season for a date

## Usage

``` r
iata_season_for_date(date)
```

## Arguments

- date:

  a date

## Value

the name of the IATA season in the form `summer-yyyy`

## Examples

``` r
if (FALSE) { # \dontrun{
season_iata("2024-04-01")
} # }
```
